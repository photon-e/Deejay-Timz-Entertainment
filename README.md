# Deejay Timz Entertainment Ltd

Premium Next.js 15 website for DJ services, event sound systems, entertainment, DJ academy, booking, gallery, admin dashboard, SEO, Prisma/PostgreSQL, NextAuth, and Cloudinary-ready media management.

## Deploy to Netlify
Set `DATABASE_URL`, `ADMIN_EMAIL`, `ADMIN_PASSWORD`, and `NEXT_PUBLIC_SITE_URL`, then run `npm run build`.
